# Shore — SaaS HR Platform Landing Page

A responsive landing page for an HR SaaS platform built with React and Tailwind CSS.

## 🚀 Live Demo
[View Live](https://komal05-web.github.io/SaasApp-LandingPage-using-React-TailwindCSS)

## ⚙️ Getting Started

```bash
git clone https://github.com/komal05-web/SaasApp-LandingPage-using-React-TailwindCSS.git
cd SaasApp-LandingPage-using-React-TailwindCSS
npm install
npm run dev
```

## 🛠️ Tech Stack
- React 18
- Tailwind CSS
- Vite
- Lucide React
- shadcn/ui

## ✨ Features
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll navigation
- Hero, Benefits, Pricing, Testimonials, FAQ sections
- Footer with social links
